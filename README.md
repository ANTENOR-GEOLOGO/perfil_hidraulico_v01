# Perfil Hidráulico - QGIS Plugin (v1.1)

Plugin especializado para el post-procesamiento de datos hidráulicos en QGIS. Permite integrar Modelos Digitales de Elevación (DEM) con ejes de río para el análisis técnico de secciones transversales.

## 🛠️ Capacidades Técnicas
* **Cálculo Hidráulico:** Obtiene Área (A), Espejo de Agua (T), Perímetro Mojado (P) y Radio Hidráulico (Rh).
* **Análisis de Flujo:** Cálculo automático del **Número de Froude (Fr)** y clasificación del régimen (Subcrítico, Crítico, Supercrítico).
* **Visualización:** Gráficos en **msnm** de topografía y niveles de agua con soporte para escala 1:1.

## 🚀 Instalación
1. Descarga este repositorio.
2. Copia la carpeta `perfil_hidraulico` en la ruta de plugins de QGIS:
   `AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins`
3. Reinicia QGIS y activa el complemento.
